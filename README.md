<h2>🎮 Sobre o jogo</h2>

<p>
<b>TIME HUNT</b> é um jogo de plataforma 2D desenvolvido na Unity utilizando C#.
O jogador é colocado em um mundo noturno e hostil onde o tempo está constantemente se esgotando.
</p>

<p>
Para sobreviver, é necessário derrotar inimigos, explorar o mapa e coletar recursos que aumentam o tempo restante.
</p>

<hr/>

<h2>🧠 Conceito</h2>

<p align="center">
<b>⏳ Tempo = Vida</b>
</p>

<p align="center">
Cada decisão impacta diretamente na sua sobrevivência.
</p>

<hr/>

<h2>📜 Prompt / História</h2>

<p>
Você acorda em um mundo corrompido onde o tempo foi fragmentado.
Criaturas emergem das sombras, e apenas derrotando-as você consegue recuperar fragmentos temporais.
Seu objetivo é sobreviver tempo suficiente para encontrar todas as chaves e escapar.
</p>
